#  ContentCrafter AI

AI-powered content repurposing web application that converts long-form text into:

* LinkedIn posts
* Twitter/X threads
* Instagram captions
* Relevant hashtags

using LLM-based AI generation.

---

## Tech Stack

### Frontend

* React.js
* Vite
* CSS

### Backend

* Node.js
* Express.js

### AI

* Groq API
* Llama 3.1

---

##  Features

* AI-powered content generation
* Tone customization
* Responsive UI
* Copy-to-clipboard support
* Platform-specific content formatting

## ⚙️ Setup

### Backend

```bash
cd backend
npm install
node server.js
```

Create `.env` file:

```env 
GROQ_API_KEY=your_api_key_here
PORT=5000
```

### Frontend

```bash 
cd frontend
npm install
npm run dev
```
